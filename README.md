# CP Quera Solutions

A curated repository of programming-problem solutions from [Quera](https://quera.org/), organized by problem name and implemented mainly in **Python** and **C++**.

This repository is intended for learning, practice, and solution review. Each problem folder usually contains the original problem statement as a PDF and one or more accepted/working source-code implementations.

## Repository contents

- **103 Quera problem folders**
- **103 PDF problem statements**
- **74 Python solution folders/files**
- **67 C++ solution folders/files**
- Additional language samples: **3 Java** and **1 C#** where available

## Folder structure

Most problems follow this structure:

```text
problem-name/
├── problem-name.pdf      # Problem statement
├── py/
│   └── main.py           # Python solution
└── cpp/
    └── main.cpp          # C++ solution
```

Some older folders may contain extra IDE/build artifacts such as `.exe`, `.o`, `.class`, `.cbp`, or `.layout` files. These files are not required to run the solutions and can be ignored or removed from future commits.

## How to run a solution

### Python

```bash
python "problem-name/py/main.py"
```

### C++

```bash
g++ -std=c++17 "problem-name/cpp/main.cpp" -o solution
./solution
```

Most solutions read input from **standard input** and write output to **standard output**, matching the format expected by Quera's online judge.

## Requirements

### Python

- Python 3.8+

No external Python packages are required for the current solutions unless a specific problem folder states otherwise.

### C++

- A C++17-compatible compiler, such as `g++` or `clang++`
